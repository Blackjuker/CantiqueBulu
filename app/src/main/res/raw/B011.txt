11
JOE DOE E NE ETYI
Ch.H. 64

1
Jôé dôé é ne étyi,
Etyi ya nnôm éto. 	          )
Duma, étôtok bi n'émbiôé, ) bis
Zambe ya nnôm éto	          )

2 
Bi tele asu dôé,
A kômbô kaf'ie wo. 	         )
Nyoñe duma bia zu ve wo, ) bis
Zambe Nté yôp a si              )

3
Womien kobô'ô bi wôk ! 
Bi ne momo a Tat. 		)
Ve bia kômbô same minlem, ) bis
Bi yen étôto'o jôé. 		)

4 
Jôm ése é vee si nyô,
A bi ya yôp été, 	    )
Beta'ane metyiñ a kañe ) bis
Nyô a to nnôm éto !	    )
